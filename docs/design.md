# Vadgäller – Design v1

## Stil
- Ljust tema, Google/ChatGPT-känsla
- Mobil först
- Maxbredd 720px på desktop
- Stor input, tydliga chips, minimalt brus

## Layout
- Header: företagsnamn (valfri logga)
- Content: chat
- Footer/input: sticky längst ner

## PIN-sida
- Centrerad
- 4 rutor
- Auto-submit när 4 siffror

## Chat
- Tomt läge: input + chips
- Svar i cards + källa-rad
- Inga menyer i anställdvy
